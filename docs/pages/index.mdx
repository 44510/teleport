Teleport provides connectivity, authentication, access controls and audit for
infrastructure. Teleport understands the SSH, HTTPS, RDP, Kubernetes API, MySQL,
MongoDB and PostgreSQL wire protocols, plus many others. It can integrate with
Single Sign-On providers and enables you to apply access policies using
infrastructure-as-code and GitOps tools.

## Get started

- Try out Teleport by signing up for a [14-day free
  trial](https://goteleport.com/signup).
- For more information about the value Teleport provides for securing your
  infrastructure, read [What is Teleport?](./get-started/what-is-teleport.mdx).
- Understand the components of a Teleport cluster by reading [How Teleport
  Works](./get-started/how-teleport-works.mdx).
- Follow a tutorial to learn how to protect your first server with Teleport.
  [Try Out Teleport](./get-started/try-out-teleport.mdx). (It is also possible
  to self-host your Teleport cluster.)
