<Details title="How to tell if TLS Routing is Enabled" opened={false}>

<ScopedBlock scope={["oss", "enterprise"]}>
To get details of the Teleport Cluster you can invoke `/webapi/ping` path
on the proxy service.  The `proxy.tls_routing_enabled` value provides if TLS routing is enabled.
Here's an example for the proxy on `https://teleport.example.com`:

```code
$ curl https://teleport.example.com/webapi/ping |  jq '.proxy.tls_routing_enabled'
true

```
</ScopedBlock>
